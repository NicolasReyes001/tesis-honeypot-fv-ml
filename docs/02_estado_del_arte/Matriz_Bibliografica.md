# Convención de Identificadores

Cada referencia tendrá un identificador único con el siguiente formato:

[EJE]-[AÑO]-[NÚMERO]

Ejemplo:

E2-2024-001

donde:

- E2 → Eje de investigación
- 2024 → Año de publicación
- 001 → Consecutivo dentro del eje y año
