## Fase 1.

### ¿Qué es la identidad de un proyecto de investigación?

 Conjunto de características que definen de forma precisa a un proyecto, de manera resumida y practica muy usado en investigación ya que se usa como base para la construcción del proyecto.

### ¿Qué elementos debe contener?

La identidad del proyecto debe contener los siguientes componentes o elementos:

- Título provisional: Delimita el alcance del proyecto.
- Área de conocimiento: Ubica el proyecto dentro de la Ingeniería Electrónica.
- Linea de investigación: Relaciona el trabajo con la línea institucional.
- Campo de aplicación: Define el contexto donde se aplicará la solución.
- Tipo de investigación: Clasifica el enfoque metodológico del proyecto.
- Enfoque metodológico: Define cómo se obtendrán y analizarán los resultados.
- Modalidad del proyecto: Especifica el tipo de trabajo desarrollado.
- Producto esperado: Describe el entregable principal.
- Tecnologías principales: Resume las herramientas y plataformas empleadas.
- Palabras claves: Facilitan la identificación temática del proyecto.
